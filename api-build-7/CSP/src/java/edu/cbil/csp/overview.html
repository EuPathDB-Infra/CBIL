A Java port of CBIL's CSP, Version 0.1

Please see the package overview files {@link cbil.csp} and {@link cbil.csp.dialog} for a description of how to use these class libraries.

Note to CBIL users:

To use these packages ({@link cbil.csp} and {@link cbil.csp.dialog}) you must include the file /usr/local/lib/java_classes/JCSP.jar in your Java CLASSPATH.
