Formating

These methods generate formatting HTML.

ep
es
These are utilities routines which are used to make other routines easier to code. They break from the standard argument passing model for ease of use.

First argument is $args containing values from caller\'s caller. Next argument is element tag.

Remining arguments are attribute names which can be inserted if specified.

es is for single tag elements, like C<<HR>>.

ep is for paired tag elements like C<<font>...</font>>. ep can be prevented from including the beginning or ending tags by defining BEGIN = 0> or END = 0> in the arguments. This is handy for long lazy contents.
