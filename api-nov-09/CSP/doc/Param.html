Backward Compatibility of Tags

Here are the conversions from the non-oo CSP tags to current tags.

Name -> name which is inherited from item.

Prompt -> text which is inherited from item.

Optional -> optional

Defaults -> defaults

Range -> range

Length -> length

Type -> type
