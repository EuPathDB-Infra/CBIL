Description

Configuration

Methods

new
creates a new topic.

Arguments:

item arguments as well as these:

addItem
adds an item to a topic.

Arguments:

items is a reference to a list of items to add.

makeDisplay
adds HTML for display of topic and items under it.

Arguments:

prefix text to put before each item.

suffix text to end each item.

makeHelp
adds HTML help for topic and items under it.
