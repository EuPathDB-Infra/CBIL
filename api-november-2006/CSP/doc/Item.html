Item

NAME

csp::item - basic dialog element item.

DESCRIPTION

Inherits from csp::html so that any item can be used to create HTML easily.

ATTRIBUTES

which helps answer questions about style.

name which is used to refer to the item.

text bit of text which every item can use to display itself.

depth depth in hierarchy. Generally used to guide indentation. Defaults to 0 when an item is created, then is adjusted when it is added to a topic.

help bit of help text.

seealso list of other items of interest.

METHODS

Create and Utility Methods

new
creates a new item.

Arguments:

req style_config

req name

req text

req help

opt seealso

HTML-Generating Methods

helpTarget
creates the anchor tag HTML for the help section.

Arguments:

viz the stuff that is visible in the help link.

helpLink
creates the reference tag HTML for a link to the help section of an object.

Arguments:

viz the stuff that is visible in the help link.

formTarget
creates the anchor tag HTML for the form section.

Arguments:

viz the stuff that is visible in the form link.

formLink
creates the reference tag HTML for a link to the form section of an object.

Arguments:

viz the stuff that is visible in the form link.

seeAlso
adds 'See Also's HTML for the help section.
