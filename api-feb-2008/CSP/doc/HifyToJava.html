HifyToJava

Name

CSP::HifyToJava - generate a Java(TM) version of CSP::Hify

Description

This package performs a single task; it generates a .java source file that is equivalent (more or less) to CSP::Hify.pm

The generated Java file is called HTMLUtil.java and it is accompanied by a utility class, AH.java, that provides a convenient way of creating Hashtables from even-length arrays. AH is the Java equivalent (more or less) of CSP::B It is used to store (HTML attribute, value) pairs.

See the script ``hifytojava'' for an example of how to invoke this package.

Methods

generateJavaHify
Generate the Java(TM) source code for Hify.

First argument is the name of the target Java(TM) package.

Second argument is the path of the corresponding directory.

generateHandlerMethod
All requests on the HTMLUtil object are forwarded to the Java(TM) handler methods generated by this Perl method.

generateTagMethods
Generates all the methods in the HTMLUtil class that correspond to a particular HTML element/tag type (e.g. BR, TABLE, TR, etc.)

generateClassHeader
Generates the initial section (package statement, comments, class inheritance declarations) of a .java file.

generateClassFooter
Generates the final section of a .java file.

generateArrayHashtableClass
Generates the utility class ``AH''; this class extends java.util.Hashtable and allows new instances to be initialized from even-length arrays.

generateElementClass
Generates a utility class, Element, that is used to represent an HTML element/tag and the allowed attributes for that tag. The definition of the Element class is local to the HTMLUtil.java file.
